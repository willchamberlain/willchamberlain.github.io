VRep script parameters

Summary

Vrep script parameters are LUA variables which can be set before or during a simulation without changing the LUA script code. The parameter values can be changed by double-clicking on the bar-chart icon of the script.

Each script parameter is owned by a particular script.

Parameters can be added to a script outside a simulation by double-clicking on the bar-chart icon of the script.

Parameters can be added to a script during a simulation programmatically - this is a form of data exchange between scripts, or to/from remote API.

API

API : simGetScriptSimulationParameter( [handle of object that owns the parameter] , [parameter name string] )

API : simSetScriptSimulationParameter( [handle of object that owns the parameter] , [parameter name string] , value )

(all parameters are treated as strings, but can be converted to number later on. Strings may contain any values (also embedded zeros)) : see the data packing/unpacking functions.

Usage

Get:
ultrasound_check_rate_hz = simGetScriptSimulationParameter(sim_handle_self,"ultrasoundCheckRate")
